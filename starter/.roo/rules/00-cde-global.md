# CDE Global Operating Rules (All Modes)

**Context is binding.** Context documents are normative constraints, not suggestions.

If documented intent is unclear, incomplete, or contradictory, you must **stop** and surface the issue explicitly.  
Silence in the face of ambiguity is a failure mode.

**AGENTS.md is authoritative** for all AI agents operating in this repository.

If multiple instructions conflict, the **more restrictive** instruction takes precedence.
